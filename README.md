# Ayush Kabra — Portfolio Website

A modern, minimalist portfolio website showcasing my work as a Product Manager.

## Features

- Clean, modern design with custom cursor interactions
- Responsive layout for all devices
- Smooth scroll animations
- Sections: About, Experience, Projects, Ideas, Contact

## Technologies

- Pure HTML, CSS, and JavaScript
- No frameworks or dependencies
- Custom animations and interactions
